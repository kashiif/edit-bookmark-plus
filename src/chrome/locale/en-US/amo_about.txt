<ol>
<li><b>Re-size Bookmark Popup</b>: By default, the popup that appears when you click on yellow star in the address bar, is fixed size. This add-on makes it re-sizable and expands bookmarks tree to take remaining space. You can resize the popup to a size of your own choice.

<li><b>Always Show the Selected Folder</b>: Sometimes the bookmarks tree does not scroll to the folder which is selected from Folder drop down list. This issue is fixed by this add-on.

<li><b>Option to Disable Popup Autohide</b>: Add/Edit bookmark closes automatically as soon as you clicked anywhere on the page. This makes it harder to copy/paste text from web page to any field of popup. This behavior can be disabled from the options dialog of the addon.

<li><b>Option to Show/Hide Extra Fields</b>: You can customize whether to show description, location and keywords fields on popup or not.

<li><b>Update Name and/or url of a bookmark easily</b>: Adds 'Update Url' and 'Update Name' menu items in the context menu of bookmarks toolbar, bookmarks menu, bookmarks sidebar. Both of the menu items can be turned off by Edit Bookmark Plus options dialog.

<b>Update Url</b>: Updates the url of selected bookmark with the url of current page.

<b>Update Name</b>: Updates the name (the text that shows up on UI) of selected bookmark with the title of current page.

</ol>
